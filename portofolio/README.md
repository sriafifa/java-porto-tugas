# Website Portfolio Sri Afifa

## Deskripsi Proyek
Website portfolio pribadi untuk Sri Afifatul Fajriya, mahasiswa semester 4 Universitas Hasyim Asy'ari, Program Studi Sistem Informasi. Website ini menampilkan informasi pribadi, keahlian, portofolio proyek, dan informasi kontak.

## Fitur Utama
1. **Header & Navigasi**
   - Judul "Welcome to Portfolio Sri Afifa"
   - Menu navigasi dengan 5 bagian: Home, About, Skills, Portfolio, Contact
   - Responsif dengan hamburger menu untuk perangkat mobile

2. **About Section**
   - Foto profil
   - Deskripsi pribadi dan latar belakang pendidikan
   - Detail informasi tentang Sri Afifatul Fajriya

3. **Skills Section**
   - Menampilkan keahlian utama dengan indikator persentase
   - Animasi progress bar yang interaktif
   - 4 kategori keahlian: Desain, Web Development, Database, Responsive Design

4. **Portfolio Section**
   - Menampilkan 5 proyek dengan filter kategori
   - Setiap proyek memiliki deskripsi dan teknologi yang digunakan
   - Efek hover dan animasi

5. **Contact Section**
   - Informasi kontak: email, telepon/WhatsApp, Instagram, lokasi
   - Form kontak yang dapat mengirim pesan
   - Validasi form dan notifikasi

6. **Footer**
   - Informasi hak cipta
   - Link sosial media
   - Quick links ke semua bagian website

## Teknologi yang Digunakan
- **HTML5**: Untuk struktur halaman
- **CSS3**: Untuk styling dengan berbagai jenis selektor
- **JavaScript**: Untuk interaktivitas dan efek dinamis
- **Font Awesome**: Untuk ikon
- **Google Fonts**: Untuk tipografi

## Persyaratan CSS yang Dipenuhi
1. **Selektor Elemen**: Digunakan untuk styling tag HTML standar (p, h1, h2, a, dll)
2. **Selektor ID**: Minimal 3 ID untuk bagian utama (#header, #home, #about)
3. **Selektor Kelas**: Lebih dari 5 kelas untuk komponen berulang (.highlight, .btn, .section-title, .skill-item, .portfolio-item, dll)
4. **Selektor Universal**: Untuk reset margin/padding dasar pada semua elemen
5. **Pengelompokan Selektor**: Untuk mengelompokkan elemen yang memiliki style sama

## Fitur JavaScript
1. **Navigasi Responsif**: Hamburger menu untuk perangkat mobile
2. **Scroll Effects**: Header sticky, back-to-top button, active link pada scroll
3. **Portfolio Filter**: Filter proyek berdasarkan kategori
4. **Animasi Skill**: Progress bar dengan animasi dan counter
5. **Form Validation**: Validasi form kontak dengan notifikasi
6. **Typing Effect**: Efek mengetik pada judul hero section
7. **Notification System**: Sistem notifikasi untuk feedback user
