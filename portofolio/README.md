# Portfolio Sri Afifa

Ini adalah website portfolio pribadi **Sri Afifatul Fajriya**, mahasiswa Semester 4 Universitas Hasyim Asy'ari, Prodi Sistem Informasi. Website ini menampilkan informasi profil, keterampilan, proyek, dan kontak.

## Fitur

- **Responsive Design**: Tampilan optimal di semua perangkat (desktop, tablet, mobile).
- **Navigasi Sticky**: Header tetap terlihat saat scroll.
- **Hamburger Menu**: Untuk tampilan mobile.
- **Filter Portfolio**: Menyaring proyek berdasarkan kategori.
- **Smooth Scroll**: Navigasi yang halus ke setiap bagian.
- **Back to Top Button**: Tombol untuk kembali ke atas.
- **Animasi Scroll**: Elemen muncul saat discroll.
- **Form Kontak**: Simulasi pengiriman pesan.

## Teknologi yang Digunakan

- HTML5
- CSS3 (Flexbox, Grid, Animasi)
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts
